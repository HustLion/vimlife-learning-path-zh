# 前言

这本书的内容是Hustlion的对于编程学习路线的思考，同时包括Hustlion非常重视的理念、技巧等，将其总称为VimLife。

于是有了"VimLife码农成长指南"这个名字。

内容来源：
1. 自己的思考；
2. 看到别人的文章、公众号推送后，觉得很好，拿过来结合自己的想法进行重写；

打算写中文、英文版。
